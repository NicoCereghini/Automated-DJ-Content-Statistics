Privacy Policy

Last updated: [02-20-2026]

This application ("Automated DJ Content Statistics") is a private analytics tool used solely by the account owner to retrieve performance data from TikTok and Instagram through their official APIs.

Data Collection

The application only accesses analytics data from the authenticated TikTok and Instagram accounts. No personal data from other users is collected.

Data Usage

All retrieved data is used exclusively for personal analytics and performance tracking. The data is not shared, sold, transferred, or distributed to third parties.

Data Storage

Analytics data may be stored locally on the account owner's system for reporting and analysis purposes. No data is stored on external servers beyond the official platform APIs.

Third-Party Services

This application interacts only with official APIs provided by:

  • TikTok
  • Meta (Instagram Graph API)
